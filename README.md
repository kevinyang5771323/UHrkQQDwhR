# 医院挂号预约系统 java413

## 项目概述

本项目是基于Java语言，采用SpringBoot与Vue.js的前后端分离架构开发的医院挂号预约管理系统。它包含了丰富的功能模块，能够满足医院日常挂号、预约、信息管理等多种需求。

## 技术栈

- **前端技术**：Vue2 + ElementUI
- **后端技术**：SpringBoot2 + MyBatisPlus + Redis
- **数据库**：MySQL
